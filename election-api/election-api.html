/results/{election_type}/{year}

/results/{election_type}/{year} get

get

get /results/{election_type}/{year}

  • Request
  • Response

URI Parameters

  • election_type: required (string)

    Election type

  • year: required (integer)

    Valid election type

HTTP status code 200

Election results for given election type and year

Body

Type: application/json

Example:

{
"request": {
  "election_type": "parliament_bih",
  "year": 2010
},
"municipality_results": [
  {
    "name": "Velika Kladusa",
    "id": 1,
    "electoral_units": [
      {
        "id": "511",
        "results": [
          {
            "name": "Party #1",
            "abbreviation": "p1",
            "votes": 5
          },
          {
            "name": "Party #2",
            "abbreviation": "p2",
            "votes": 2
          }
        ]
      }
    ]
  }
]
}

HTTP status code 404

Missing election data
